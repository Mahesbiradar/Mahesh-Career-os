# SQL-OS

> An independent, self-contained learning system for SQL.
> No other subject. No global roadmap. Open this folder and study.

## What's Inside

| File / Folder | Purpose |
|---|---|
| `STATUS.md` | Your current position, topic progress, and submission log |
| `SYLLABUS.md` | Full SQL syllabus |
| `RESOURCE_MAP.md` | Every syllabus topic mapped to a specific resource |
| `PROMPTS.md` | Three prompts — Daily Lesson, Evaluation, Notes Update |
| `KNOWLEDGE_RULES.md` | How SQL concepts should be organized in `notes/` |
| `notes/` | Your SQL knowledge base — one file per topic |
| `lessons/` | Daily lesson files generated by the agent |
| `submissions/` | Your submitted SQL queries and written work |

## How to Use Daily

**Morning (5 min)**
1. Open `PROMPTS.md` → Copy Prompt 1 → paste into AI
2. Agent reads `STATUS.md` + `RESOURCE_MAP.md` → generates `lessons/DAY_N.md`
3. Open the lesson, watch the video, do the assignment

**After the session**
4. Copy Prompt 2 → paste into AI → paste your work below it
5. Agent evaluates and updates `STATUS.md`

**After evaluation (optional)**
6. Copy Prompt 3 → AI updates `notes/`

| Grade | Meaning | What Happens Next |
