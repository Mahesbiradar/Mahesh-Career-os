# Submissions Folder

Save your assignment files here before or after pasting them in chat.

## Naming Convention

```
day01_variables.py
day02_operators.py
day03_control_flow.py
day08_sql_joins.sql
day12_os_processes.md      ← theory topics saved as markdown
day15_http_summary.md
```

## Not Required

You do not have to save files here. Pasting code directly in chat is enough.
This folder is optional — use it if you want a local record of your work.
