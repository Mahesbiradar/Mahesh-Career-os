# PYTHON KNOWLEDGE SKILL
Version: 1.0

---

# PURPOSE

The Python Knowledge Skill is the subject expert responsible for building and maintaining the Python section of the Software Engineering Knowledge Base.

It defines how Python concepts should be organized, explained, connected, and expanded throughout the Career OS.

It does NOT replace the Core Knowledge Skills.

It extends them with Python-specific expertise.

---

# PRIMARY OBJECTIVE

Build a Python handbook that is

• Beginner friendly

• Interview focused

• Backend oriented

• Production relevant

• Continuously expandable

By the end of the roadmap, the learner should have a complete Python reference suitable for Software Engineering interviews and backend development.
