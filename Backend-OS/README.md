# Backend-OS

> An independent, self-contained learning system for Backend Engineering.
> No other subject. No global roadmap. Open this folder and study.

---

## What's Inside

| File / Folder | Purpose |
|---|---|
| `STATUS.md` | Your current position, topic progress, and submission log |
| `SYLLABUS.md` | Full Backend Engineering syllabus |
| `RESOURCE_MAP.md` | Every syllabus topic mapped to a specific resource. Agent uses this. |
| `PROMPTS.md` | Three prompts — Daily Lesson, Evaluation, Notes Update |
| `KNOWLEDGE_RULES.md` | How Backend concepts should be organized and explained in `notes/` |
| `notes/` | Your Backend knowledge base — one file per topic, growing as you study |
| `lessons/` | Daily lesson files generated by the agent |
| `submissions/` | Your submitted code and written work |

---

## How to Use Daily

**Morning (5 min)**
1. Open `PROMPTS.md`
2. Copy **Prompt 1 — Daily Lesson** → paste into AI chat
3. Agent reads `STATUS.md` + `RESOURCE_MAP.md` → generates `lessons/DAY_N.md`
4. Open the lesson, watch/read the resource, do the assignment

**After the session**
5. Copy **Prompt 2 — Evaluation** → paste into AI → paste your work below it
6. Agent evaluates and updates `STATUS.md`

**After evaluation (optional but recommended)**
7. Copy **Prompt 3 — Notes Update** → paste into AI
8. Agent updates the relevant file in `notes/`
