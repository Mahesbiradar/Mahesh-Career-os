# Mahesh Career-OS

A self-directed, structured learning system for building strong software engineering fundamentals — Python, backend engineering, SQL, databases, operating systems, computer networks, frontend development, cloud deployment, and communication skills, all tracked day by day.

## Preview

This repo is a living curriculum rather than an app, so the "preview" is the roadmap itself:

- **01_MASTER_SKILL_TREE.md** — the full map of skills being built
- **15_180_DAY_ROADMAP.md** — the day-by-day execution plan
- **STATUS.md** — current progress log, updated as each day is completed
- **lessons/** and **submissions/** — daily notes and completed exercises

## What's Inside

- `Career-OS/Python` — Python lessons and practice (variables, data types, and beyond)
- `Career-OS/backend` — backend engineering notes and exercises
- `Career-OS/communication` — communication-skills practice
- `Career-OS/lessons` — daily lesson logs
- `Career-OS/submissions` — completed daily submissions
- Numbered syllabus files (`02`–`10`) covering Python, Backend Engineering, SQL, DBMS, Operating Systems, Computer Networks, Frontend, Cloud & Deployment, and Communication
- `11_JOB_PREPARATION_SKILL_TREE.md`, `12_MASTERY_FRAMEWORK.md`, `13_SYSTEM_SIMPLIFICATION_RULES.md` — supporting frameworks for how the learning is structured
- `16_EXECUTION_ENGINE.md`, `17_WEEK_1_SPRINT.md`, `18_CAREER_OS_OPERATOR_GUIDE.md` — how the plan is executed week to week
- `19_COURSE_ENGINE.md`, `20_RESOURCE_MAP.md`, `PROMPTS.md` — curated resources and prompts used along the way
