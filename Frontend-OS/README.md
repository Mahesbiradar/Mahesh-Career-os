# Frontend-OS

> An independent, self-contained learning system for Frontend Engineering.
> Covers HTML, CSS, JavaScript, and React.

## What's Inside
| File / Folder | Purpose |
|---|---|
| `STATUS.md` | Current position, topic progress, submission log |
| `SYLLABUS.md` | Full Frontend Engineering syllabus |
| `RESOURCE_MAP.md` | Every topic mapped to a specific video |
| `PROMPTS.md` | Three prompts — Daily Lesson, Evaluation, Notes Update |
| `KNOWLEDGE_RULES.md` | How Frontend concepts should be organized in `notes/` |
| `notes/` | Your Frontend knowledge base — one file per topic |
| `lessons/` | Daily lesson files |
| `submissions/` | HTML/CSS/JS/React code and written work |

## How to Use
**Morning:** Prompt 1 → lesson → watch video → build/write
**After session:** Prompt 2 + submission → agent evaluates → updates STATUS.md
**After evaluation:** Prompt 3 → notes/ update
