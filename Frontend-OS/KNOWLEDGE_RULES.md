## TEACHING STYLE
- Problem → User experience → Browser behaviour → Code → Backend connection
- Never start with syntax
- Always explain WHY the browser works this way
- Connect every React concept to the request-response lifecycle

## QUALITY CHECK
Failed if: taught as syntax only, no browser explanation, no backend connection.
Succeeds when: learner understands how browser renders, how React updates UI, and how it connects to backend APIs.
