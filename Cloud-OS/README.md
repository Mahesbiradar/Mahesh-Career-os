# Cloud-OS

> An independent, self-contained learning system for Cloud & Deployment.
> Covers Git, Docker, Linux, CI/CD, Nginx, and AWS basics.

## What's Inside
| File / Folder | Purpose |
|---|---|
| `STATUS.md` | Current position, topic progress, submission log |
| `SYLLABUS.md` | Full Cloud & Deployment syllabus |
| `RESOURCE_MAP.md` | Every topic mapped to a specific course/video |
| `PROMPTS.md` | Three prompts — Daily Lesson, Evaluation, Notes Update |
| `KNOWLEDGE_RULES.md` | How Cloud concepts should be organized in `notes/` |
| `notes/` | Your Cloud knowledge base — one file per topic |
| `lessons/` | Daily lesson files |
| `submissions/` | Config files, scripts, written explanations |

## How to Use
**Morning:** Prompt 1 → lesson → watch/read → hands-on practice
**After session:** Prompt 2 + submission → evaluates → updates STATUS.md
**After evaluation:** Prompt 3 → notes/ update
