# DBMS-OS

> An independent, self-contained learning system for Database Management Systems (DBMS).
> Theory-focused subject. Lessons are written explanations, not code.
> Open this folder and study independently.

## What's Inside
| File / Folder | Purpose |
|---|---|
| `STATUS.md` | Current position, topic progress, submission log |
| `SYLLABUS.md` | Full DBMS syllabus |
| `RESOURCE_MAP.md` | Every topic mapped to a Gate Smashers video |
| `PROMPTS.md` | Three prompts — Daily Lesson, Evaluation, Notes Update |
| `KNOWLEDGE_RULES.md` | How DBMS concepts should be organized in `notes/` |
| `notes/` | Your DBMS knowledge base — one file per topic |
| `lessons/` | Daily lesson files |
| `submissions/` | Written theory submissions (.md files) |

## How to Use Daily
**Morning:** Paste Prompt 1 → agent generates lesson → watch Gate Smashers video → write explanation
**After session:** Paste Prompt 2 + your written submission → agent evaluates → updates STATUS.md
**After evaluation:** Paste Prompt 3 → agent updates notes/
