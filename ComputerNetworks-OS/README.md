# ComputerNetworks-OS

> An independent, self-contained learning system for Computer Networks.
> Theory-focused subject. Lessons are written explanations following packet flow.

## What's Inside
| File / Folder | Purpose |
|---|---|
| `STATUS.md` | Current position, topic progress, submission log |
| `SYLLABUS.md` | Full Computer Networks syllabus |
| `RESOURCE_MAP.md` | Every topic mapped to a Gate Smashers video |
| `PROMPTS.md` | Three prompts — Daily Lesson, Evaluation, Notes Update |
| `KNOWLEDGE_RULES.md` | How CN concepts should be organized in `notes/` |
| `notes/` | Your CN knowledge base — one file per topic |
| `lessons/` | Daily lesson files |
| `submissions/` | Written theory submissions (.md files) |

## How to Use
**Morning:** Prompt 1 → lesson → watch video → write explanation
**After session:** Prompt 2 + submission → agent evaluates → updates STATUS.md
**After evaluation:** Prompt 3 → notes/ update
